Générateur de Script pour Déploiement GitHub Pages

Avant de Commencer : Préparation de Votre Projet Local

L'outil "Générateur de Script" ne crée PAS le dossier du projet ni son contenu sur votre Mac. Il vous aide à initialiser Git dans un dossier que vous avez déjà préparé, à le lier à GitHub, et à pousser son contenu.

Étapes préliminaires SUR VOTRE MAC :

  • Créez le dossier pour votre nouveau projet : Par exemple, dans votre dossier Documents/ProjetsGitHub/, créez un nouveau dossier nommé presentation-Codev (ou le nom de votre projet).
  • Placez tous les fichiers de ce projet DANS ce dossier : Cela inclut votre fichier index.html principal, vos fichiers style.css, script.js, vos images, et tous les autres fichiers ou sous-dossiers qui constituent ce projet spécifique.

Une fois votre dossier de projet local prêt :

  1. Remplissez les champs ci-dessous :
    • Champ "Chemin complet..." : Entrez le chemin exact vers le dossier que vous venez de créer et de remplir (ex: /Users/davidpaucot/Documents/ProjetsGitHub/presentation-Codev).
    • Remplissez les autres champs (nom du dépôt GitHub, utilisateur GitHub, message de commit).
  2. Cliquez sur "Générer le Script et les Instructions".
  3. Suivez ensuite les instructions générées (Partie 1, Partie 2, Partie 3).

Cet outil vous aidera à générer les commandes pour publier un nouveau projet sur GitHub Pages. Remplissez les champs ci-dessous après avoir préparé votre dossier local comme indiqué ci-dessus.

Partie 1 : Création du Dépôt sur GitHub.com

  1. Allez sur https://github.com/new pour créer un nouveau dépôt.
  2. Repository name : Entrez "". N'utilisez pas d'espaces dans ce nom. Utilisez des tirets ou underscores.
  3. Assurez-vous que "Public" est sélectionné.
  4. NE COCHEZ AUCUNE CASE d'initialisation ("Add a README file", "Add .gitignore", "Choose a license"). Le dépôt doit être créé complètement vide.
  5. Cliquez sur "Create repository".
  6. Sur la page suivante (celle avec "Quick setup... or create a new repository on the command line"), repérez et copiez l'URL HTTPS. Elle ressemblera à :
    https://github.com//.git
    Le script terminal ci-dessous utilisera cette URL (elle y sera déjà insérée).

Partie 2 : Script à Copier-Coller dans votre Terminal

Ouvrez votre Terminal (ou Git Bash sur Windows). Assurez-vous d'être dans votre dossier "home" (~) ou un dossier parent de votre projet (pas à l'intérieur du dossier du projet lui-même) avant de coller, car la première commande du script est cd "CHEMIN_DU_PROJET".

Copiez TOUT le bloc de commandes ci-dessous et collez-le en une seule fois dans votre terminal. Appuyez sur Entrée. Le script s'exécutera étape par étape. Surveillez les messages.
